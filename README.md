# Gxib

*build tool for the gxi-rs project*

## Install

```bash
cargo install gxib
```

## Usage

Run in project root dir.

*Build for desktop*

```bash
gxib desktop
```

*Build for web*

```bash
gxib web
```

## Other Args

Run help to list other commands and args

```bash
gxib help
```
